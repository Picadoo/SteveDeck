# 截图清单

截好按文件名放本目录，README 占位自动生效。统一深色主题、隐藏真实服务器 IP / 令牌。

| 文件名 | 内容 | 要求 |
|---|---|---|
| `viewer.png` | 3D 实时视角页 | 桌面宽屏；画面里有聊天字幕悬浮最佳（README 首图） |
| `inventory.png` | 背包界面 | 打开背包，最好正在拖拽一个物品 |

| `modules.png` | 托管模块面板 | 开着 1-2 个模块、带运行统计（挖矿数 / 击杀数） |
| `scripts.png` | 积木脚本编辑器 | 有一个配置好的触发器 + 动作 |
| `mobile.png` | 手机浏览器 | 竖屏，连上引擎后的界面 |

宽图（viewer / modules / scripts）建议 1280×800 左右；mobile 竖屏原始分辨率即可。PNG 体积大可转 JPG / WebP，README 里同步改后缀。
